# 常规操作
```bash
# 步骤1：确认当前分支
git branch
* master
(切换到 master 分支 git checkout master )

# 步骤2：同步远程分支信息
git fetch origin

# 步骤3：对比差异（简洁模式）
git diff --name-status master origin/master

#步骤4：拉取到本地
git pull 

#步骤5：本地化开发


#步骤6： 添加所有变动文件（按需调整扩展名）
git add .

# 步骤7：提交更改
git commit -m "提交版本的说明"

# 步骤8： 推送到远程
git push origin master

```
